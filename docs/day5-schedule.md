# 📅 DAY 5｜培訓課表

# DAY 5：整合實作＋外場＋總考核

## 課程時間表

### 第一節｜洗牌／射牌動作調整（30 分鐘）

**對應教材**

- [DAY1｜洗牌（對應教材）](/day1-shuffling)
- [DAY1｜射牌（對應教材）](/day1-pitching)

**最終確認**：動作穩定度與流暢性

---

### 第二節｜細部規定（30 分鐘）

**對應教材**

- [DAY4｜錯誤發牌／有效行動（對應教材）](/day4-misdeal)
- 第二章 → 2. 玩家加入遊戲（延遲報名／候補／重購）
- 第二章 → 3. 離座與手牌保留
- 第五章 → 1. 錯誤發牌／有效行動發生

**學習內容**（依課表）

1. **加入遊戲判定**
    - 小盲與按鈕位之間不可立即領牌
    - 延遲報名的座位安排
2. **離座處理**
    - 離座時間限制
    - 手牌保留規則
3. **錯誤發牌**
    - 何時可宣告錯發
    - 有效行動產生後不得回溯

---

### 第三節｜模擬練習（30 分鐘）

**練習內容**

- 完整賽事模擬（開桌到結束）
- 培訓師扮演玩家，製造各種情境
- 新人獨立處理所有狀況

---

### 第四節｜外場工作（30 分鐘）

**對應教材**

- [DAY5｜外場工作](/day5-floor-work)
- 第八章 → 外場工作流程（完整操作細節）

**學習重點**（依課表）

1. **點碼、Key 編號**
    - 賽事結束後立即點碼並 Key 入表格
    - 確認結餘為 0（詳細操作見第八章 → 3、4）
2. **收位卡、確認人數**
    - 截買後 15～20 分鐘執行 Key ID
    - 收回位卡回報櫃檯（詳細操作見第八章 → 4）
3. **拆桌、併桌**
    - 判斷時機與平衡桌數
    - 協助玩家轉移（詳細流程由培訓師現場示範）

---

### 第五節｜洗牌／射牌動作調整（30 分鐘）

**最終調整**：為總考核做準備

---

### 第六節｜總考核（30 分鐘）

**考核項目**

1. **基礎技能**
    - 洗牌、射牌流暢度
    - 點碼速度與準確度
2. **規則判定**
    - 級距計算
    - 主邊池分配
    - 玩家行動判讀
3. **異常處理**
    - 錯誤發牌
    - 犯規應對
    - 何時呼叫 Floor
4. **綜合表現**
    - 桌面節奏控制
    - 荷官禮儀
    - 專業態度

**及格標準**：各項目皆達 80 分以上

---

## 當日重點提示

**外場工作核心要點**

> 詳細操作流程請參閱：第八章 → 外場工作流程、[DAY5｜外場工作](/day5-floor-work)
> 

**點碼作業**（第八章 → 3）

- 賽事結束立即執行
- Key 入表格確認結餘為 0

**Key ID 作業**（第八章 → 4）

- 截買後 15～20 分鐘執行
- 攜帶脆盒、依座位詢問 ID、收回位卡

**拆桌併桌**

- 觀察人數、判斷時機
- 平衡各桌、協助轉移

**環境維護**（第八章 → 7）

- 收碼、理牌、清潔
- 點碼優先度大於理牌

## 當日檢核表

### 必備技能確認

- [ ]  洗牌、射牌動作標準且流暢
- [ ]  點碼快速且準確（60 顆 5 分鐘內完成）
- [ ]  級距計算正確率 80% 以上
- [ ]  主邊池分配無失誤
- [ ]  能判讀所有常見牌型
- [ ]  了解錯誤發牌與有效行動規則
- [ ]  能正確應對常見犯規
- [ ]  知道何時應呼叫 Floor
- [ ]  完成外場工作流程（點碼、Key ID、收位卡）
- [ ]  態度積極、禮貌專業

### 培訓結果判定

✅ **通過培訓**：發放培訓津貼 3000 元，安排正式排班

❌ **未通過培訓**：可能給予補考機會/不予錄用

## 培訓師總結

> **給培訓師的提醒**
> 

> - 最後一天要給予新人信心，但也要確實把關
> 

> - 觀察新人是否具備「獨立上桌」的能力
> 

> - 記錄新人的強項與弱項，供後續排班參考
> 

> - 通過培訓的新人，建議前 3～5 班安排資深荷官帶班

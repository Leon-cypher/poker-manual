## 📋 基本處置原則

::: tip
⚖️

**違規處置標準流程**：

1. **第一次犯規** → 荷官口頭警告一次
2. **第二次犯規** → 荷官立即叫 Floor，裁判到場後 **Ban 該玩家一手牌**
3. **第三次犯規** → 裁判 **Ban 該玩家三手牌**

**重要**：身為裁判與荷官，首要任務是**維持遊戲秩序**，同時協助新手理解規則，並在必要時制止惡意犯規。目標是讓玩家能在良好的環境下競技，維持遊戲的**公平**與**公正**。

:::

---

## 🚦 違規行為的分類

### 輕微違規（可先提醒）

::: tip
⚠️

這類行為通常是玩家不熟悉規則或一時疏忽，**可以先友善提醒**，若再犯再叫 Floor。

:::

| 違規行為 | 說明 | 荷官提醒話術 |
| --- | --- | --- |
| **越位行動（Out of turn）** | 玩家在尚未輪到自己行動時就做出下注相關動作 | 「先生您好，還沒輪到您，請稍等」 |

---

### 嚴重違規（立即叫 Floor）

::: tip
🔴

這類行為影響遊戲公平性或秩序，**必須立即叫 Floor**，不得自行處理。

:::

| 違規行為 | 說明 | 為何嚴重 |
| --- | --- | --- |
| **角牌** | 與其他玩家合謀作弊 | 嚴重違反公平原則 |
| **語言攻擊** | 辱罵其他玩家或荷官 | 破壞遊戲氛圍 |

---

## 🛠️ 違規處理標準流程

### 流程圖

```
發現違規行為
   ↓
【判斷嚴重程度】
   ↓
輕微違規？
   ├─ 是 → 友善提醒
   │         ↓
   │      玩家配合？
   │         ├─ 是 → 繼續遊戲
   │         └─ 否 → 叫 Floor
   │
   └─ 否（嚴重違規）
         ↓
    【立即叫 Floor】
         ↓
    保持冷靜，說明狀況
         ↓
    執行 Floor 指示
```

## 📞 如何正確「叫 Floor」

### 叫 Floor 的時機

::: tip
✅

**必須叫 Floor 的情況**：

1. 玩家之間發生爭執
2. 規則判定有疑問（如複雜的主邊池）
3. 發生嚴重違規行為
4. 錯誤發牌且不確定如何處理
5. 玩家要求叫 Floor（玩家有權要求裁判介入）
:::

### 叫 Floor 的標準話術

::: tip
🗣️

**荷官統一用語**：

「Floor,桌號」

**音量要求**：清楚但不急躁，讓附近的 Floor 能聽到

:::

---

### Floor 到場後的荷官職責

#### 1. 簡潔說明狀況

::: tip
💬

**好的說明範例**：

「4 號座位推出籌碼後又收回，5 號座位認為應該算有效行動」

「輪到 3 號玩家行動，5 號玩家越位下注」

**不好的說明**：

「他們兩個在吵架」（太籠統）

「我覺得應該是...」（不要加入個人判斷）

:::

#### 2. 客觀描述，不加評論

- ✅ **只說發生了什麼**
- ❌ **不說你覺得誰對誰錯**

#### 3. 執行 Floor 的指示

- Floor 做出判決後，荷官依指示執行
- 即使你認為判決有問題，也不要當場質疑
- 若有疑問，可在該手牌結束後私下詢問

## 🎲 情境案例

### 案例 1：越位行動（輕微違規）

**狀況**：

- 翻牌圈，輪到 A（小盲位）行動
- A 正在思考期間，B 直接下注 1500（越位行動）

**處理原則**：

先觀察 A 的行動，再決定如何處置 B 的越位行動。

**處理步驟**：

1. **立即制止 B**：「先生您好，還沒輪到您」
2. **等待 A 完成行動**
3. **判定 B 的越位行動是否有效**：

::: tip
⚖️

**判定規則**：

**情況 1：A 做出攻擊行為（下注、加注）**

→ B 的越位行動**不必強制保留**

→ B 可在輪到自己時**重新決定動作**

→ 對 B **口頭警告一次**

**情況 2：A 沒有做出攻擊行為（過牌、棄牌）**

→ B 必須**維持原本的越位動作**（視同有效動作保留）

→ 對 B **口頭警告一次**

:::

1. **若 B 是第二次犯規**：立即叫 Floor，執行 Ban 一手牌處分
2. **提醒話術**：「請注意桌況，避免越位影響其他玩家。這次先警告一次，之後再發生會依規定請 Floor 處理。」

---

### 案例 2：底牌曝光（輕微違規）

**狀況**：

- 玩家 A 在 Flop 圈思考時，不小心把底牌翻得太高
- 玩家 B 看到了其中一張牌

**處理**：

1. **友善提醒**：「先生您好，請保護好您的手牌」
2. **若玩家 B 提出異議**：
    - 叫 Floor
    - 說明「A 的底牌被 B 看到」
3. Floor 通常不會處分（非故意），但可能口頭警告，並公開B玩家看到的牌給所有人知道

---

### 案例 3：語言攻擊（嚴重違規）

**狀況**：

- 玩家 A 輸掉一手牌後，開始辱罵玩家 B
- 使用髒話和攻擊性語言

**處理**：

1. **立即叫 Floor**：「Floor, please!」
2. **保持冷靜**，不要跟著情緒化
3. **向 Floor 說明**：「A 號座位對 B 號座位使用了不當語言」
4. **等待 Floor 處理**：可能口頭警告、罰坐一輪、甚至取消比賽資格

::: tip
⚠️

**荷官不得參與玩家爭執**

即使你覺得某位玩家很沒禮貌，也不要加入爭論。你的角色是「中立的遊戲執行者」，交給 Floor 處理。

:::

---

### 案例 4：推出籌碼後反悔（嚴重違規）

**狀況**：

- 玩家 A 推出 10000 籌碼
- 玩家 B 立刻跟注
- 玩家 A 突然說「等等，我要收回」

**處理**：

1. **立即叫 Floor**
2. **說明狀況**：「A 號推出籌碼後想收回，B 號已經跟注」
3. Floor 通常會裁定：
    - **籌碼已推出 + 手已縮回** → 有效行動，不得反悔
    - **籌碼還在手上** → 可能允許修正

## 📋 快速檢核表（SOP）

**發現違規時**

- [ ]  判斷嚴重程度（輕微 vs. 嚴重）
- [ ]  輕微違規 → 友善提醒
- [ ]  嚴重違規 → 立即叫 Floor

**友善提醒階段**

- [ ]  使用禮貌用語（「先生您好」「請」「謝謝」）
- [ ]  清楚說明問題（「還沒輪到您」「請明確表達意圖」）
- [ ]  若玩家不配合 → 叫 Floor

**叫 Floor 時**

- [ ]  清楚呼叫「Floor, please」
- [ ]  簡潔說明狀況（客觀描述，不加評論）
- [ ]  等待 Floor 到場，保持現場狀態

**Floor 處理中**

- [ ]  不要插話或加入個人意見
- [ ]  執行 Floor 指示
- [ ]  若有疑問，該手牌結束後再詢問

## ⚠️ 常見錯誤與修正

| 錯誤 | 正確做法 |
| --- | --- |
| ❌ 自己判定嚴重違規的處分 | ✅ 立即叫 Floor，不要自己決定 |
| ❌ 提醒時語氣不佳 | ✅ 永遠保持禮貌與專業 |
| ❌ 跟玩家爭論規則 | ✅ 簡單說明後，不確定就叫 Floor |
| ❌ Floor 到場後加入個人意見 | ✅ 只做客觀描述，不評論 |

## 💬 老手提醒

::: tip
🧠

**「先提醒，再叫 Floor」是原則，但不是鐵律**

新手最容易犯的錯是：

1. **太客氣**：明明是嚴重違規，還在猶豫要不要叫 Floor

2. **太強硬**：輕微違規就立刻叫 Floor，讓氣氛緊張

記住：

- **輕微違規 + 初犯 = 友善提醒**
- **嚴重違規 / 再犯 / 不確定 = 立即叫 Floor**

你的首要任務是「讓遊戲順利進行」，其次才是「維護規則」。保持冷靜、禮貌、中立，是荷官最重要的素養。

:::

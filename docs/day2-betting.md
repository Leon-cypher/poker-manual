## 🎯 核心概念：下注判定的三大原則

::: tip
💡 **記住這三句話：**

1. **口頭說的算數** — 說了就要照做，籌碼只是執行
2. **過 50% 算加注意圖** — 超過基準點就要補到最小加注
3. **一次動作原則** — 第一次推出去的才算，不能反悔
:::

---

## 📋 判定流程圖

```
玩家推出籌碼
     ↓
有口頭宣告嗎？
     ↓
   有 → 以口頭為準（籌碼只是執行）
     ↓
   沒有 → 看推出幾顆籌碼？
     ↓
【單顆籌碼】→ 判定為跟注（無論面額多大）
     ↓
【複數籌碼】→ 移除一顆最小面額
     ↓
   剩餘金額足夠跟注嗎？
     ↓
   不夠 → 判定為跟注
     ↓
   足夠 → 計算「目前下注額 + 級距 × 50%」
     ↓
   推出金額 ≥ 基準點 → 判定為加注，補到最小加注
   推出金額 < 基準點 → 判定為跟注
```

## 🔥 情境案例：桌上會遇到的狀況

### 案例 1：單顆大額籌碼（最常見）

**情境：**
- 玩家 A 下注 1500
- 玩家 B **沒說話**，推出一顆 5000

**判定：**
✅ **跟注 1500**（單顆籌碼 = 跟注，無論面額多大）

**找零：**
- 先從底池拿 3500 給玩家 B
- 再將 5000 收入底池

---

### 案例 2：複數籌碼 — 剛好跟注

**情境：**
- 玩家 A 下注 2400
- 玩家 B 推出 2 個 1000 + 1 個 500

**判定流程：**
1. 移除最小面額（500）→ 剩 2000
2. 2000 < 2400（不足跟注）
3. ✅ **判定為跟注 2400**

---

### 案例 3：複數籌碼 — 加注意圖（重點）

**情境：**
- 玩家 A 下注 1500
- 玩家 B 推出 1 個 5000 + 1 個 500

**判定流程：**
1. 移除最小面額（500）→ 剩 5000
2. 5000 > 1500（足夠跟注）
3. 計算基準點：1500 + 1500 × 50% = **2250**
4. 5000 > 2250 → **有加注意圖**
5. 最小加注 = 1500 + 1500 = **3000**
6. 玩家實際推出 5500 > 3000
7. ✅ **判定為加注 5500**

## ❌ 常見錯誤 & 地雷區

::: warning ⚠️ 新手最容易搞錯的 5 件事：
1. **看到多顆籌碼就直接算總額** → 正確：先移除一顆最小面額再判斷
2. **單顆 5000 跟注 1500，以為是加注** → 正確：單顆籌碼 = 跟注
3. **超過跟注額就算加注** → 正確：要超過「下注額 + 級距 × 50%」才算
4. **玩家說完 Raise 又說 Call** → 正確：以第一次宣告為準
5. **All-in 9000 就能讓後面的人加注** → 正確：增量 < 級距 = 關閉
:::

## 💡 老手提醒：實戰經驗

::: tip 🧠 實戰提醒
**提醒 1：找零先給，再收籌碼**
正確流程：1. 先拿找零 2. 讓玩家確認 3. 再收籌碼。

**提醒 2：看不清楚就請玩家重新擺放**
「請您將籌碼重新擺放，讓我確認一下。」

**提醒 3：口頭宣告要聽清楚**
沒聽清楚就問：「請再說一次？」

**提醒 4：級距計算不用當場算給玩家聽**
簡單回答：「依規則需補到最小加注。」保持節奏。
:::

## 🧪 練習題：測測你學會了嗎？

### 題目 1：基礎判定
盲注 200/400，玩家 A 下注 1000，玩家 B 推出 2 個 1000
::: details 💡 點擊查看答案
**答案：跟注 1000**
1. 移除最小面額（1000）剩 1000
2. 1000 = 跟注額
3. 判定為跟注
:::

### 題目 2：50% 基準點
玩家 A 下注 2000，玩家 B 推出 1 個 5000 + 1 個 1000
::: details 💡 點擊查看答案
**答案：加注 4000**
1. 移除最小面額（1000）剩 5000
2. 基準點 = 2000 + 1000 = 3000
3. 5000 > 3000 → 有加注意圖
4. 最小加注 = 4000
:::

### 題目 3：口頭 vs 籌碼
玩家 A 下注 3000，玩家 B 說「Raise 8000」，但推出 1 個 10000
::: details 💡 點擊查看答案
**答案：加注 8000**
口頭宣告優先於籌碼。找零 2000 給玩家 B。
:::

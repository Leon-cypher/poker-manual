## 📍 座位與位置名詞

### 三個關鍵位置

| 位置 | 英文 | 職責 | 行動順序特性 |
| --- | --- | --- | --- |
| **按鈕位** | BTN (Button) | 標記發牌起/終點 | Preflop 倒數第三個行動
翻牌後**最後行動** |
| **小盲注** | SB (Small Blind) | 自動投入半個 BB | Preflop 倒數第二個行動
翻牌後**第一個行動** |
| **大盲注** | BB (Big Blind) | 自動投入完整 BB | Preflop **最後行動**
翻牌後第二個行動 |

::: tip
💡

**記憶口訣**：

- Preflop：BTN → SB → BB（最後行動）
- 翻牌後：SB → BB → ... → BTN（最後行動）
- **翻牌後位置優勢翻轉，BTN 變最有利**
:::

### UTG（Under The Gun）

- **定義**：BB 順時針方向第一位，Preflop **第一個行動**的玩家
- **壓力最大**：還沒看到任何人的反應就要決定，最容易被加注

## 🔄 完整遊戲流程（五個階段）

::: tip
⏭️

**進入下一階段的核心條件**（荷官必須確認）：

1. ✅ 所有仍在牌局中的玩家**下注面額一致**
2. ✅ 所有玩家都已完成行動（All-in 玩家除外，無需再行動）
3. ✅ 沒有人可以再行動（無人待補注、無人待決定）

**簡單判斷法**：當所有人面前的籌碼數量相同，且輪了一圈回來，就可以進入下一階段。

:::

### 階段 0：發底牌前

1. **收集盲注**
    - 荷官確認 SB 和 BB 已投入場中
    - SB 通常是 BB 的一半（例：BB 2000 → SB 1000）
2. **發底牌**
    - 從 SB 開始，順時針每人發一張
    - 發完一輪後，再發第二張
    - 每人最終持有 **2 張底牌**（面朝下）

---

### 階段 1：Preflop（翻牌前）

::: tip
🎲

**此時桌面上沒有任何公共牌**

:::

#### 行動順序

```
UTG → UTG+1 → UTG+2 → ... → BTN → SB → BB（最後行動）
```

#### 常見情境

| 情境 | 說明 | 結果 |
| --- | --- | --- |
| 所有人 Fold 到 BB | BB 自動贏得底池（SB + BB） | 直接結束該手牌 |
| UTG 加注，其他人跟注 | 行動輪到 BB，BB 可選擇跟注/加注/棄牌 | BB 是 Preflop 最後決定權 |
| 有人 All-in | 其他人依序決定是否跟注 | 可能產生邊池 |

#### 🚦 何時進入 Flop？

::: tip
✅

**確認以下條件後，才能發 Flop**：

- 所有仍在牌局的玩家下注面額相同（例如全部都投入 5000）
- BB 已經行動過（BB 是 Preflop 最後決定者）
- 如果 BB 後面有人加注，要等到所有人都跟注/棄牌/All-in

**例**：UTG 加注到 5000，所有人跟注 5000，BB 也跟注 5000 → 可以發 Flop

**例**：UTG 加注到 5000，BTN 再加注到 15000，所有人跟注 15000 → 可以發 Flop

:::

---

### 階段 2：Flop（翻牌圈）

::: tip
🃏

**荷官發出 3 張公共牌**（面朝上）

:::

#### 流程

1. **燒牌**：荷官從牌堆頂拿一張牌面朝下丟棄
2. **發三張公共牌**：一次性翻開 3 張牌，排成一列
3. **下注輪開始**：

```
SB → BB → UTG → ... → BTN（最後行動）
```

#### 🚦 何時進入 Turn？

::: tip
✅

**Flop 下注輪結束條件**：

- 所有玩家下注面額一致
- BTN（或其前一位活躍玩家）已完成行動
- 如果有人下注/加注，所有其他人都已跟注/棄牌/All-in

**例**：SB Check、BB Check、UTG 下注 3000、BTN 跟注 3000、SB Fold、BB 跟注 3000 → 可以發 Turn

:::

::: tip
⚠️

**位置優勢翻轉**：翻牌後 SB 變成第一個行動，BTN 變成最後行動（最有利）

:::

---

### 階段 3：Turn（轉牌圈）

::: tip
🃏

**荷官發出第 4 張公共牌**

:::

#### 流程

1. **燒牌**
2. **發第四張公共牌**：加在 Flop 三張牌的右側
3. **下注輪**：順序同 Flop

```
SB → BB → UTG → ... → BTN
```

#### 🚦 何時進入 River？

::: tip
✅

**Turn 下注輪結束條件**：

- 所有玩家下注面額一致
- 最後行動者（通常是 BTN）已完成行動
- 條件判定方式與 Flop 相同
:::

---

### 階段 4：River（河牌圈）

::: tip
🃏

**荷官發出第 5 張（最後一張）公共牌**

:::

#### 流程

1. **燒牌**
2. **發第五張公共牌**：加在 Turn 牌的右側
3. **最後下注輪**：順序同前

```
SB → BB → UTG → ... → BTN
```

#### 🚦 何時進入 Showdown（攤牌）？

::: tip
✅

**River 下注輪結束條件**：

- 所有玩家下注面額一致
- 最後行動者已完成行動
- 剩餘 2 人以上 → 進入攤牌
- 只剩 1 人 → 直接贏得底池，無需攤牌
:::

---

### 階段 5：Showdown（攤牌）

#### 攤牌規則

::: tip
👁️

**核心原則**：最後一個**主動下注/加注**的玩家先翻牌

:::

| 情境 | 誰先翻牌 |
| --- | --- |
| River 有人下注，其他人跟注 | **下注者先翻** |
| 所有人 Check 到底 | **BTN 左邊第一位先翻**（通常是 SB） |
| 多人 All-in 無後續行動 | **最後加注者先翻** |

#### 比牌與發放籌碼

1. **荷官唸出牌型**：「Three of a kind, Kings」
2. **比較大小**：依標準牌型大小（同花順 > 四條 > 葫蘆 > ...）
3. **推籌碼給贏家**：
    - 單一贏家：全部推給他
    - 平手（Chop）：底池平分
    - 有邊池：先處理主池，再依序處理邊池

## 🎲 情境案例

### 案例 1：Preflop 全棄到 BB

**狀況**：

- 6 人桌，UTG ~ BTN 全部 Fold
- SB 也 Fold
- 只剩 BB

**處理**：

1. 荷官收走所有棄牌
2. 將 SB + BB 的籌碼推給 BB
3. 該手牌結束，不發公共牌

---

### 案例 2：翻牌後 SB 先說話

**狀況**：

- Preflop 後剩 SB、BB、BTN 三人
- Flop 發出 K♠ 9♥ 3♦

**行動順序**：

```
SB 先行動 → BB → BTN（最後）
```

**實況**：

- SB：Check
- BB：下注 5000
- BTN：Fold
- 行動回到 SB，SB 可選擇 Call / Raise / Fold

---

### 案例 3：River 攤牌順序

**狀況**：

- River 牌面 A♠ K♠ Q♠ 10♥ 2♣
- BTN 下注 10000
- SB 跟注 10000
- BB Fold

**攤牌順序**：

1. **BTN 先翻**（因為他是最後下注者）
2. BTN 翻出 J♠ 9♠（同花順）
3. SB 可選擇翻牌或直接棄牌（Muck）
4. 荷官推籌碼給 BTN

---

## ⚠️ 常見錯誤與修正

| 錯誤 | 正確做法 |
| --- | --- |
| ❌ 翻牌後讓 UTG 先行動 | ✅ 翻牌後永遠是 SB 先（或 SB 左邊第一位活躍玩家） |
| ❌ 攤牌時讓 SB 先翻 | ✅ 最後下注/加注者先翻 |
| ❌ 發 Flop 時忘記燒牌 | ✅ **每次發公共牌前都要燒一張** |
| ❌ 把 BTN 當成 Dealer | ✅ BTN 只是位置標記，荷官是工作人員 |

## 🎯 快速檢核表（SOP）

**發底牌前**

- [ ]  確認 SB / BB 已投入
- [ ]  從 SB 開始發牌，每人兩張

**Preflop**

- [ ]  從 UTG 開始行動
- [ ]  BB 是最後行動者

**Flop / Turn / River**

- [ ]  每次發牌前**燒一張**
- [ ]  從 SB（或其左邊）開始行動
- [ ]  BTN 最後行動

**Showdown**

- [ ]  最後下注者先翻牌
- [ ]  無下注則 BTN 左邊第一位先翻
- [ ]  唸出牌型並拉出需要用到的公牌然後推籌碼給贏家

🧠

新手荷官最容易搞混「Preflop vs. 翻牌後」的行動順序。記住：

- Preflop：BB 有優勢（最後說話）
- 翻牌後：BTN 有優勢（最後說話）

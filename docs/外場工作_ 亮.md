# 外場工作_ 亮

外場工作核章：「核心概念」

德州店家營運離不開「賽事」的開啟和結算，營收的來源100%為「服務費」。而外場工作的核心概念為「推動賽事進行」一切「牌桌外的行政事務」都為外場工作的範圍，使賽事進行地流暢，迅速，準確率高。

工作範圍：

離開「荷官」位時，所有額外的行政工作都為外場工作的範圍，分為「與賽事有關」和「與店家秩序」有關。

工作角色與態度：

「外場」「牌桌」「櫃檯」是截然不同的工作內容，但是卻又緊緊相依的關係，外場的角色即為仲介/銜接者，需要讓櫃檯與牌桌產生連結的重要角色，現場狀況瞬息萬變，外場人員應隨時保持清晰的思緒，實時掌控現場狀況的變化，即時完成當下未完成之事務。需要對當前賽事進行的程度有一定的理解。

個人事務與工作的取捨：

如遇不可抗之生理需求，（想上廁所/抽菸），應與場控或櫃檯清晰交接去向後才可離開崗位，並且現場調度與個人事務相衝突時，應以現場狀況為主，聽從現場調度。

工作內容:

外場工作

攤牌／計時器

有使用的桌子不一定會有牌，至櫃台或其他尚未使用的桌子取用紅藍兩幅牌呈半圓形攤放即可。

常見問題

1. 沒有剩餘的牌可供更換：櫃台下有常備全新的牌可供使用，拆封後即可使用
2. 桌上的牌過於磨損：同上

計時器

至櫃台內取用充電完成的計時器，並更替未充電之計時器。

常見問題

1. 計時器螢幕過暗：至櫃台更換新的計時器，將螢幕亮度過低之計時器進行充電即可
2. 計時器時間不符合 30 秒：同時按紅藍鍵直到螢幕閃爍為止，閃爍後即可按紅藍鍵進行調整，調整至 30 秒後再次同時按紅藍鍵即可完成調整。

換電視

電視呈現「紅色底」或是與櫃台即將開出之賽事不符時，於櫃台抽屜中選擇正確號碼的遙控器進行更換即可

遙控器使用說明：

按 ← 鍵回到總頁面，按 ↓ 找到更新鍵，更新後找到正確賽事即可。

常見問題

1. 網路不穩定導致無法更換至正確畫面：回報至櫃台即可
2. 遙控器沒電：櫃台內有電池可供更換，和櫃台回報即可
3. 找不到正確賽事：詢問櫃台是否有正確開啟賽事，如尚未開啟賽事，等候櫃台通知即可

點碼

賽事結束（螢幕呈現紅色底）時，開啟共享表格「限時錦標賽計算」，照「位子」清點籌碼後 key 入表格，直至表格結餘為 0 為止。點完所有籌碼後回報給櫃台「XX 賽事好了」即可進入收碼環節

如何擺放？

1. 依單位：十萬、五萬、萬、千、百，每「十個」為一組（柱），每五「柱」為一排（橫向）
2. 如遇到（5000）（500）時，每「2 個」為一組（柱），每五柱為一排（橫向）
3. 維持一柱的計量單位為萬／千（20 個 5000 可為一柱，即十萬）
4. 如遇到超過 20 個單位想進位為一柱時：例 7 個 1000、30 個 100 時，小碼在下，大碼在上，以頂端一柱為代表，並與其他相同單位平行。

常見問題

1. 總籌碼結餘不符合 0：詢問櫃台正確組數，組數無誤的話換人複查，差距在 1000 以下即可回報櫃台。
2. 總數差距過大且複查多次：回報櫃台即可。

記編號

賽事截至買入後，剩餘 15～20 分鐘時，至櫃台攜帶相對應之脆盒，打開共享表格「限時錦標賽計算」，找到相對應的賽事後依座位順序詢問客人 ID，key 入所有 ID 後放置脆盒於桌上空位，收回位卡至櫃台回報。

常見問題

1. 玩家不在位子上：詢問桌上其他玩家是否認識，回報人名至櫃台／詢問櫃台 ID
2. 沒有位子放盒子：使用周邊推車放置即可
3. 兩桌以上賽事：12／一桌 3 個、34／一桌 3～4 個、66／一桌 4 個、11 以上／一桌 4～5 個
4. 由兩桌併成一桌之賽事：依級別再加兩個盒子即可

收碼

大於 5000 之籌碼獨立一柱（20 顆），小於 5000 之籌碼依 10 個 100、2 個 500、8 個 1000 為一柱為單位收起（20 顆）。大於 3 盒請分批收回櫃台，避免打翻。

常見問題

急需使用桌子時，可先全數放入盒子後轉移到別桌，再行整理即可。

理牌

賽事結束後，當桌荷官需將牌整理為黑桃 A 最上，依序為 ♠ → ❤ → ♣ → ♦，可先將花色區分開後再行處理。

常見問題

人手短缺時，牌桌優先度為點碼大於理牌，桌上荷官應優先幫助點碼後再行理牌。

環境維護

離開牌桌且無其他事務時，主動清掃之事項

1. 邊車：賽事結束時，將所有垃圾／飲料集中至一車後推至洗手間分類處理
2. 室外：休息區桌面應慣性常備煙杯各 1，超過 ⅔ 之煙杯即應更換。煙柱一個月清理一次即可
3. 停車區以及延伸路面（環繞店家部分）有垃圾和煙蒂應清掃完全
4. 室內：垃圾桶超過 7 分滿即可更換新垃圾袋。閉店前應使用吸塵器清潔全店，吸完地面後至洗手間調和拖地水（自來水和地板清潔劑比例為 20／1），先拖牌桌周邊，最後拖走道和櫃台內即可。
5. 牌桌桌面：櫃台內有常備滾筒和刷子，沿桌邊進行清理桌面。
6. 牌桌桌沿／荷官椅／玩家椅：洗手間內常備皮革清潔劑，搭配對應顏色抹布，適量噴灑於抹布上進行清潔即可。
7. 洗手間：於閉店前拉出所有坐廁的垃圾袋，使用潔廁劑進行清潔後，使用水管和馬桶刷清潔即可。（週末時倒小通，並通知早班進行整理）
8. 洗手台和玻璃：使用菜瓜布和潔廁劑進行清潔即可

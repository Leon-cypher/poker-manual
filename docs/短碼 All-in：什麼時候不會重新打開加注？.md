# 短碼 All-in：什麼時候不會重新打開加注？

下次複習: 2026年1月26日
優先順序: TDA補充
出處/備註: TDA Rule 47；教材第三章 6.3 口訣「增量 vs 級距」
學習順序: 125
已掌握: No
正面（問題）: 短碼 all-in 造成的增量不足時，為什麼後面已行動過的玩家不能再加注？
背面（答案）: 在 NL/PLO 中：短 all-in 的增量 < 本輪最後合法級距（最後合法加注增量）時，不會重新打開加注；已行動過的玩家回到行動時只能跟注或棄牌。
若多個短 all-in 的累積增量達到完整級距，才可能重新打開。

類別: 下注/級距

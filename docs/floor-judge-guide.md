# 如何成為一個好的裁判

熟悉規則只是基本功；能在第一時間掌握現場狀況，並做出讓玩家信服的判決，才是一位好的裁判。

---

### 第一步：快速到場

遇到荷官呼叫裁判時，第一件事就是**立刻前往**。

在玩家眼中，若裁判看起來懶散或慢吞吞地走過去，容易留下不專業的印象；反之，**迅速到場並展現專業形象**，更能提升後續判決的說服力。

### 第二步：釐清現況（先把資訊問完整）

到場後，裁判應先聽荷官說明遇到的問題，並**引導對方把所有爭議點與關鍵資訊說清楚**，以便做出正確判決。

- 先確認：事件發生在哪個階段？牽涉到哪些動作？是否已有人行動？
- 再確認：桌上是否有人提出異議？各方說法是否一致？

**情境示例：** 荷官說「翻牌時不小心忘記燒牌怎麼辦？」

這類情況在 TDA 裡通常有標準處理方式，但裁判在裁決前必須先確認：**是否已經有玩家在該公共牌之後採取行動**。因此，關鍵不在於「忘記燒牌」本身，而在於「事件發生後，遊戲流程是否已經往下走了」。

### 第三步：做出判決，並讓玩家理解

在資訊完整後，裁判需要根據規則與現場狀況做出判決，並以清楚、簡潔的方式向玩家說明理由。

玩家可能會有不同看法，此時裁判要能**相信自己的專業判斷**，並用一致的邏輯解釋，避免引發二次爭議。

### 補充：規則未涵蓋的情境怎麼辦？

有些案例可能不在 TDA 規則明文範圍內。此時應以**公平性原則**作為出發點：

- 這個處理方式是否會對某位玩家造成不合理的不利？
- 是否能在不影響比賽秩序的前提下，最大程度還原公平？

做出決定後，也要能以同樣的公平性邏輯說服玩家接受。

### 監視器調閱：最後手段

調閱監視器畫面應視為**最後手段**。若能憑藉桌上人員的記憶與資訊把情況還原，裁判應盡力先在現場釐清。

限時錦標賽中，若還原狀況確實會耗費時間，可以**暫停比賽時間**以完成釐清與裁決。

請謹記：

- 不是玩家要求就暫停。
- 是否暫停，應由裁判在評估後判定「確有必要」才執行。
